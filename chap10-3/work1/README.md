▶STL 라이브러리에서 컨테이너 클래스, 이터레이터 클래스, 알고리즘 함수의 역할을 각각 설명하시오.  

컨테이너: 데이터를 저장하는 자료 구조를 구현한 템플릿 클래스

이터레이터: 컨테이너의 원소들을 접근하기 위한 클래스, 컨테이너 원소에 대한 포인터를 저장하는 클래스  

알고리즘: 컨테이너 원소에 대한 복사, 검색, 삭제, 정렬 등을 구현한 템플릿 함수. 컨테이너의 멤버 함수가 아니고 전역함수로 구현된다.  

컨테이너, 이터레이터, 알고리즘은 항상 함께 사용된다.

▶포인터와 iterator의 공통점과 차이점은 각각 무엇인가?  
공통점    
역참조 가능, 증가/감소 연산자 사용 가능, 비교 연산 가능  
차이점   
포인터는 주소값 자체를 저장하는 변수이고, 이터레이터는 포인터처럼 동작하도록 만든 객체이며, 모든 STL 알고리즘에서 표준으로 사용된다  

▶vector 클래스의 erase, insert 멤버함수의 매개변수, 리턴값, 기능을 설명하시오.  
erase  
매개변수: 삭제할 위치를 가리키는 이터레이터  
리턴값: 삭제 후 삭제된 위치 다음 요소를 가리키는 이터레이터를 반환한다.  
기능: vector에서 요소를 삭제하는 함수로, 삭제된 뒤 뒤에 있는 요소들이 앞으로 이동하면서 연속성을 유지한다.  

insert  
매개변수: 삽입 위치 + 삽입할 값  
리턴값: insert 함수의 리턴값은 삽입된 원소의 포인터를 리턴한다.   
기능: vector에 새로운 요소를 삽입하는 함수로, 삽입 위치 이후의 요소들이 뒤로 이동하며 연속성을 유지한다.
